# ✊✋✌️ Rock Paper Scissors Game

📌 Project Overview

The **Rock Paper Scissors Game** is a simple Java-based console application that allows a user to play the classic hand game against the computer 🎮. This project is designed to demonstrate fundamental programming concepts in a fun and interactive way.

---

## 🎯 Objective

The main objectives of this project are:

📘 To gain practical experience in Java programming
🧠 To understand basic logic building and condition handling
🎮 To create an interactive console-based game
💻 To strengthen problem-solving and coding skills

---

## 📝 Project Description

Rock Paper Scissors is a classic game played between a user and the computer 🤖. Each player selects one of the three options:

🪨 Rock
📄 Paper
✂️ Scissors

The computer randomly generates its choice, and the winner is decided based on standard game rules.

This project includes:

🎮 User vs Computer gameplay
🔁 Option to play multiple rounds
🤖 Randomized computer moves
📊 Instant result display (Win / Lose / Draw)

The program is designed to be simple, clean, and easy to understand for beginners.
