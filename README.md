# ✊✋✌️ Rock Paper Scissors Game (Web Version)

📌 **Project Overview**

The **Rock Paper Scissors Game** is a simple and interactive web-based application that allows users to play the classic game against the computer 🎮.
This project is built using **HTML, CSS, and JavaScript**, focusing on creating a visually appealing and user-friendly interface.

---

## 🎯 Objective

The main objectives of this project are:

📘 To gain practical experience in HTML, CSS, and JavaScript
🧠 To understand DOM manipulation and event handling
🎮 To build an interactive browser-based game
💻 To improve frontend development skills

---

## 📝 Project Description

Rock Paper Scissors is a well-known hand game played between a user and the computer 🤖.

The user selects one of the three options:

🪨 Rock
📄 Paper
✂️ Scissors

The computer randomly generates its choice, and the winner is decided instantly based on game rules.

This website includes:

🎮 Button-based user interaction
🤖 Random computer choice generation
📊 Instant result display (Win / Lose / Draw)
🎨 Attractive UI with modern design

---

## 🛠️ Technologies Used

🧩 HTML – Structure of the webpage

## ⭐ Features

## 🎯 Game Rules

🪨 Rock beats Scissors
✂️ Scissors beats Paper
📄 Paper beats Rock
